Change your directory to the newly cloned repository:

```bash
cd digital_store
```

Ensure you have Node.js installed. Then, install the necessary dependencies by running:

```bash
npm install
```

### 4. Run the Project
After the installation is complete, start the development server:

```
npm run dev
```
